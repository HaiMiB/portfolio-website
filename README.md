**Technologien:** HTML, CSS, JavaScript, React, Node.js., Vite, Webhosting

**Ergebnisse:** Erhöhte Sichtbarkeit meiner Fähigkeiten und Projekte, positive Rückmeldungen von potenziellen Arbeitgebern und Kunden.
